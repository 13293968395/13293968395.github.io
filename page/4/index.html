AcWing3.完全背包问题

Posted on 2021-09-26 | In 数据结构与算法

动态规划

  • 状态表示与第一个背包问题相同
  • 转移方程
    • 01背包问题中，每个物品只可以选择一次，状态计算时就有两种情况(两种子集)
      • 装 与 不装
    • 完全背包问题, 每个物品可以选择无数次, 就有无限种情况（若干个子集）
      • 装 0 个 收益 f[i-1][j]
      • 装 1 个 f[i][j-1v[i]] + 1w[i]
      • 装 2 个 f[i][j-2v[i]] + 2w[i]
      • 装 3 个 f[i][j-3v[i]] + 3w[i]
      • 装 k 个 f[i][j-kv[i]] + kw[i] 此时 k*v[i] <= j

f[i][j] = max(f[i-1][j], f[i][j-1v[i]] + 1w[i], f[i][j-2v[i]] + 2w[i], f[i][j-3v[i]] + 3w[i], f[i][j-kv[i]] + kw[i])

将 j-v[i]带入上式 得

f[i][j-v[i]] = max(f[i-1][j-v[i]], f[i-1][j-2v[i]] + w[i], f[i-1][j-kv[i]] + k*w[i])

我们可以发现 2 式与 1 式很相似

将下面的代入上面的公式
得

f[i][j] = max(f[i-1][j], f[i][j-v[i] + w[i])
