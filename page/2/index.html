632.最小区间

Posted on 2022-03-02 | In 数据结构与算法

https://leetcode-cn.com/problems/smallest-range-covering-elements-from-k-lists/

思路：贪心

要从k个升序区间内，找到一个最小区间，使得k个列表中的每个列表至少有一个数包含在其中。

那么将问题转换, k个区间内找一个区间，使区间包含列表中的一个数 转换为 从列表中选至少一个数，组成的区间为最小区间

题目中还有一个条件：升序列表,有方向，我们就可以用多指针移动的方向，从而找到答案所寻求的方向。

4 10 15

0 9 12

5 18 22

选取第一个区间（4， 0， 5），这个区间为一种解，但不是最优解。 区间长度 （5 - 0 + 1） = 6

  • 接下来移动最小值0, 0一定不会变大， 区间可能会减小
  • 移动除最小值的其他元素， 最大值一定会变大，区间一定会增加

那么那么只移动最小元素即可

AcWing 154.滑动窗口

Posted on 2021-10-17 | In 数据结构与算法

#include
#include
#include

using namespace std;
const int N = 1000010;

int n, k;
int a[N];

int q[N];
int hh = 0, tt = -1;

int main()
{
cin >> n >> k;
for (int i = 0; i < n; i ++ ) cin >> a[i];
// 最小值
for (int i = 0; i < n; i ++ ) { // 队列维持单调，循环维持窗口

    if (hh <= tt && q[hh] < i - k + 1 ) hh++; //队首是否在窗口内部 
    while (hh <= tt && a[q[tt]] >= a[i]) tt--;  // 当前的值是否比队尾小，小的话一定是该值
    q[++tt] = i; //入队列
    
    //i - k + 1 > 0 表示窗口内的数量已经够三个
    if (i >= k-1) cout << a[q[hh]] << " "; 
}
cout << endl;

hh = 0; tt = -1;
// 最大值
for (int i = 0; i < n; i ++ ) {
    if (hh <= tt && q[hh] < i - k +1) hh++; 
    
    while (hh <= tt && a[q[tt]] <= a[i]) tt--;
    
    q[++tt] = i;
    
    if (i >= k-1) cout << a[q[hh]] << " ";
}

cout << endl;
return 0;

}
